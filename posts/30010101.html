2.语言环境安装

（1）jdk8(Java)官网下载

https://www.oracle.com/java/technologies/javase-downloads.html

选择开发工具安装到：

D:\website\languages\java\jdk1.8.0_241

更改jre安装到：

D:\website\languages\java\jre1.8.0_241

环境变量配置，桌面->此电脑->右键属性->高级系统设置->高级->环境变量,在系统变量中添加

1.变量名：JAVA_HOME，变量值：

D:\website\languages\java\jdk1.8.0_241

2.变量名：JAVA_PATH，变量值：

.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tool.jar

3.找到变量名Path，编辑，新建两个：

%JAVA_HOME%\bin
%JAVA_PATH%

4.键盘win+r输入cmd打开命令行窗口，输入

java -version
javac
java

检查java环境是否配置成功

（2）mingw(c)官网下载

https://osdn.net/projects/mingw/releases/

安装到：

D:\website\languages\mingw

在Basic Setup中选择Package右击选择“Mark for Installation”,勾选

mingw32-gcc-ada-bin
mingw32-gcc-fortran-bin
mingw32-gcc-g++-bin
mingw32-gcc-objc-bin

选择Installation->Apply Changes->Apply,完成后Close,配置用户环境变量

D:\website\languages\mingw\bin

检查配置是否成功，打开命令行win+r->cmd,输入

gcc -v

（3）php7.3._官网下载

https://www.php.net/

解压到D:\website\languages\php下，配置系统环境变量Path

D:\website\languages\php

检查配置是否成功，打开命令行win+r->cmd,输入

php -v

（4）python官网下载

https://www.python.org/downloads/

选择Customize installation,勾选Install launcher for all users;Add Python 3.8 to PATH,安装到：

D:\website\languages\python

如果未勾选Add Path,手动添加用户环境变量2条

D:\website\languages\python\Scripts\
D:\website\languages\python

检查配置是否成功，打开命令行win+r->cmd,输入

python

（5）node（js)官网下载

安装到

D:\website\languages\nodejs

选择Add to PATH，安装完成后检查配置是否成功，同理

node -v

在node安装目录下新建两文件夹node_cache和node_global

在cmd窗口输入

npm config set prefix "D:\website\languages\nodejs\node_global"
npm config set prefix "D:\website\languages\nodejs\node_cache"

这一步可能失败，就修改D:\website\languages\nodejs\node_modules\npm下的npmrc

prefix=D:\website\languages\nodejs\node_global
cache=D:\website\languages\nodejs\node_cache

添加用户变量Path

D:\website\languages\nodejs\node_global
D:\website\languages\nodejs\node_cache

添加系统变量NODE_PATH

D:\website\languages\nodejs\node_modules

安装成功

3.Git安装

安装到：

D:\website\tools\Git

选择Use Git from Git Bash only安装成功

4.nrm安装

目的：npm下载地址切换工具

1.使用cmd下载

npm install nrm -g

2.查询可用下载地址列表

nrm ls

3.切换npm下载地址

nrm use taobao

4.查看nrm ls是否在taobao前有*

5.nodemon安装

目的：nodemon是页面实时刷新，辅助node

1.使用命令行下载

npm install nodemon -g

2.执行js文件

nodemon ***.js

（2）进入pushd d:\website或cd d:\website

（3）返回上一级 cd ..

7.数据库安装

常用数据库mysql,sql-server,mongoDB,oracle

1.MongoDB官网下载

https://www.mongodb.com/

推荐镜像下载地址：

安装mongodb,选择Custom,修改路径

D:\Program Files\MongoDB\Server\4.1

取消勾选Install MongoDB Compass

双击安装mongodb-compass,安装完成后打开Start Using Compass,关闭。

第二次打开点Connect

https://mongoose.shujuwajue.com

8.安装vue脚手架

cmd安装

npm install -g @vue/cli

查看版本

vue -V

cmd 安装，基于3.x版本的脚手架创建vue项目

//1.基于交互式命令行的方式
vue create my-project
//2.基于图形化界面的方式
vue ui
//3.基于2.x旧模板
npm install -g @vue/cli-init
vue init webpack my-project

9.安装koa中间件

npm i koa-generator -g

初始化创建koa2目录

koa2 -e ***
cd ***
npm install

koa中间件官网

https://koa.bootcss.com

koa-router文档

https://github.com/alexmingola/koa-router

10.安装angular脚手架

npm install -g @angular/cli

查看版本

ng version

创建工作区

ng new my-app

运行应用

cd my-app
ng serve --open
npm run start
npm run build

创建组件

ng generate component heroes

安装typescript

npm install -g typescript typings

ng-model使用

导入 FormsModule

打开 AppModule (app.module.ts) 并从 @angular/forms 库中导入 FormsModule 符号。

app.module.ts (FormsModule symbol import)

import { FormsModule } from '@angular/forms'; // <-- NgModel lives here

然后把 FormsModule 添加到 @NgModule 元数据的 imports 数组中，这里是该应用所需外部模块的列表。

app.module.ts (@NgModule imports)

imports: [ BrowserModule, FormsModule ],

刷新浏览器，应用又能正常工作了。你可以编辑英雄的名字，并且会看到这个改动立刻体现在这个输入框上方的 <h2> 中。

11.安装react脚手架

npm install create-react-app -g

查看版本

create-react-app --version

创建工作区

create-react-app react-demo

运行应用

cd react-demo
npm install
npm start
