2.引入自定义图标库

iconfont图标库

iconfont.css放到common/icon.css

iconfont.ttf放到static/iconfont.ttf

修改icon.css

@font-face {font-family: "iconfont";
  src:url('data:application/x-font-woff2;charset=utf-8;base64,d09GMg') format('woff2'),
    url('/static/iconfont.ttf') format('truetype')
}

修改App.vue

/* 引入自定义图标库 */

animate中文网https://animate.style/#migration

命令行安装npm install animate.css --save

将animate.min.css复制到common/animate.min.css

修改App.vue

/* 引入animate动画库 */
    @import url("./common/animate.min.css");
