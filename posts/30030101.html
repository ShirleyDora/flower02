8.习题

表1：stu1
id(学号) name className age high 
表2：score
s_id id(学号) s_name(科目) score
根据姓名查成绩 张尚
select score from score
where id=(select id from stu1 where name='张尚')
求每个班级的总人数
select count(*) from stu1 group by className
根据班级查平时成绩
每个人的详细信息、科目和成绩
select * from stu1,score
where stu1.id=score.id
或者
select * from stu1 join score
where stu1.id=score.id
语文成绩考99分的学生信息
select * from stu1 s1,score s2
where s1.id=s2.id and s2.s_name='语文' and score=99

9.课后习题

子查询
实现语文成绩99的学生信息
SELECT * FROM stu1 WHERE id = (SELECT id FROM score WHERE s_name = '语文' AND score = 99);
成绩大于70的学生信息
SELECT * FROM stu1 WHERE id IN (SELECT id FROM score WHERE score > 70);

1.7案例

Spring框架

3.1. 作用
创建和管理对象。
传统的创建对象的方法是：Object obj = new Object();
使用Spring框架后：Object obj = 框架.getXXX();
即：开发者无需关注对象的创建过程，改由框架来创建，当需要对象时，通过框架获取即可。
传统的管理对象可能有：student.setName(“张三同学”);
使用Spring框架后，无需执行相关代码，通过配置，获取对象时，相关属性就已经有值了。
